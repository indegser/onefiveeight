# Interval Quiz Page

Build a quiz page for practicing scale degrees from random roots.

User flow:

- Show a prompt like "Ab의 4도는 뭐야?"
- The user does not type an answer.
- A single primary button first reveals the answer.
- After the answer is visible, the same button becomes "다음 질문" and generates another random prompt.

Scope:

- Add a Next.js App Router page at `/quiz`.
- Use random roots and scale degrees.
- Include navigation to the quiz page.
- Keep the implementation isolated from unrelated existing work.
