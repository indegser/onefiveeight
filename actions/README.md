# Repo-Local Actions

This directory is reserved for deterministic action wrappers used by the multi-agent system.

Suggested actions:
- `run-build`
- `run-lint`
- `run-typecheck`
- `run-tests`
- `run-e2e`
- `capture-screenshot`

In the MVP, the supervisor provides protocol and routing. Action wrappers can be added incrementally.
